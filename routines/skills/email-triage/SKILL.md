---
name: email-triage
description: Daily inbox triage — review new and unread emails, categorize them, summarize, draft replies for the ones that need them, and set the rest aside under a Waiting Analysis label. Use when the user wants to triage, review, or clean up their inbox.
---

Review inbox for new and unread emails. For each email:

1. Categorize into: Urgent, Action Needed, FYI, or Low Priority
2. Summarize the key point in one sentence
3. Draft a response for any emails marked Urgent or Action Needed
4. Apply a "Waiting Analysis" label to every FYI and Low Priority email so they are set aside for the monthly content analysis.
5. Mark each email as read once it has been processed.

Present results grouped by category, with Urgent items first. For drafted responses, keep the tone professional and concise. If the inbox is empty or all caught up, confirm that briefly.
